# Welcome to the Language Flash Hub!

## Features

This is a project made to help people learn languages.

Each user can create their own personal flashcard sets through copy and paste.

Then, three times of flashcards are produced for the user to practice with:
- From English to French Translation
- From French to English Translation
- French Diction to French Written

This allows for rounded development of the language.

Incorrect words are also repeated specifically when the user tries to reattempt the set.

## Tech Stack

- React with Vite
- TypeScript
- Supabase
- Tailwind CSS
- Shadcn UI
